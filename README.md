# Featured Projects

## 🚀 CampusOS *(In Progress)*

CampusOS is an evolving open-source campus infrastructure platform built to unify student achievements, events, organizations, and campus workflows into a single ecosystem.

Originally started as an Event Management Portal, the project expanded after identifying larger problems around fragmented campus systems and disconnected student data.

Now being developed under the NITRR-OSS ecosystem with a focus on collaborative development and long-term sustainability.

### Core Goals
- Centralized student profiles
- Achievement and activity tracking
- Club and organization management
- Event workflows and registrations
- Open and extensible architecture
- Student-first developer ecosystem

### Evolution

```text
Event Management Portal
        ↓
Campus Workflow Platform
        ↓
CampusOS (Open Source Ecosystem)
```

### Tech Stack
`Next.js` • `TypeScript` • `Node.js` • `PostgreSQL` • `TailwindCSS`
